Technical Architecture Deep Dive

NLP & Vector Search Implementation

# Semantic Query Processing Pipeline

1. Query Understanding
   Input: "free queer workshops this weekend near Kreuzberg"
   ↓
2. NLP Entity Extraction
   - Event Type: workshop
   - Attributes: free, queer
   - Time: this weekend → [2024-01-20, 2024-01-21]
   - Location: Kreuzberg → lat/lon + 2km radius
   ↓
3. Multilingual Embeddings (BERT-based)
   - German/English parallel processing
   - 768-dimensional vectors
   - Contextual understanding: "queer" → LGBTQ+ tags
   ↓
4. Vector Similarity Search (Faiss)
   - Approximate nearest neighbors
   - Cosine similarity threshold: 0.7
   - Real-time results in <100ms
   ↓
5. Ranking & Filtering
   - Location proximity boost
   - Temporal relevance
   - Source credibility scoring
